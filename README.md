
Technical Assessment App

Basic Vue.JS asp.net app 

- Frontend: Vue.js 3, Bootstrap 5
- Backend: ASP.NET Core Web API
- Database: Entity Framework Core In-Memory
- Testing: xUnit & Moq (unit tests)
- HTTP Client: Axios

## Setup Instructions

### Prerequisites
1. Install the **C# Dev Kit** VS Code extension
2. Install .NET 8.0 SDK and runtime (linux):
   ```bash
   sudo pacman -S dotnet-sdk-8.0
   sudo pacman -S aspnet-runtime-8.0
   ```
2. Install .Net 8,0 SDK and runtime (windows)
   https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime?cid=getdotnetcore&runtime=aspnetcore&arch=x64

### Backend Setup
No additional setup needed beyond prerequisites. The backend uses Entity Framework Core with in-memory database.

### Frontend Setup
Navigate to the client folder and install dependencies:
```bash
cd SingularSystemsTechnicalAssessment.Client
npm i
```

### Running the Application
- Start the launch full stack
